# Команды для AI-ассистента

Этот файл содержит команды, которые AI-ассистент должен выполнять при запросе пользователя.

## Команды сборки и запуска

### "Собери проект"
**Описание:** Автоматическая сборка проекта, копирование JAR в папку модов и запуск Minecraft через TLauncher.

**Действие:** Запустить скрипт `build_and_run.bat`

**Команда для выполнения:**
```bash
build_and_run.bat
```

**Альтернативные формулировки:**
- "Собери и запусти проект"
- "Собери мод"
- "Собери и установи мод"
- "Пересобери проект"
- "Собери проект и запусти Minecraft"

**Что делает скрипт:**
1. Закрывает все процессы Minecraft (Java)
2. Собирает проект через `gradlew.bat build`
3. Копирует JAR из `build/libs/steve-ai-mod-*.jar` в `%appdata%\.minecraft\mods`
4. Запускает TLauncher

---

## Дополнительные команды

### "Собери проект без запуска"
**Действие:** Запустить только сборку без копирования и запуска
```bash
gradlew.bat build
```

### "Очистить сборку"
**Действие:** Очистить папку build
```bash
gradlew.bat clean
```

---

**Важно:** При запросе пользователя "Собери проект" или аналогичных формулировках, AI должен запускать `build_and_run.bat` через терминал.
